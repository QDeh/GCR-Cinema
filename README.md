# GCR-Cinema

## Description du projet

Le projet GCR-Cinéma est une page web dynamique qui affiche les films à l'affiche et permet aux utilisateurs de consulter les descriptions détaillées des films.

## Utilisation du projet

Pour utiliser ce projet, il faut dans un premier temps vous créer un compte sur https://www.themoviedb.org/ pour demander l'attribution d'une clé API.

Une fois la clé API obtenue, il faut modifier le fichier .env.example en .env et insérer votre clé d'API à la place de "your_API_key".
